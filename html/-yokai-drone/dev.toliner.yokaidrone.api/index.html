Yokai

Interface for Yokai, which is one set of bot functions. The first thing you should do when you build bot is choose or implement Yokai. Yokai must be specified by command-line argument, then all specified Yokai is loaded by bootstrap. Each Yokai can have some Ability. You can gain and configure Abilitys by overriding init method. All implementation must have non-parameter constructor.
